# 🖥 Computer Sales and Services Management System

## 📌 Project Description

This project is a web-based application developed to manage **computer sales and service operations** efficiently. It allows customers to raise service requests and enables admins to manage products, users, and service tickets.

---

## 🚀 Features

* 👤 User Registration & Login
* 🛠 Service Request / Ticket Generation
* 📊 Admin Dashboard
* 🖥 Product Management
* 📁 Complaint Tracking System
* 💾 Database Management

---

## 🛠 Technologies Used

* Python (Django)
* HTML, CSS, JavaScript
* SQLite / MySQL
* Bootstrap

---

## 📂 Project Structure

* `manage.py` – Django project manager
* `appadmin/` – Admin module
* `appuser/` – User module
* `templates/` – HTML pages
* `static/` – CSS, JS, images
* `dbComputerService.sqlite3` – Database

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python manage.py runserver
```

## ⚠️ Note

* This project is developed for educational purposes.
* Database file can be replaced with MySQL for production use.
